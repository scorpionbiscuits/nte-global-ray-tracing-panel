What This Panel Actually Does

How It Works

The game hides ray tracing options via a GPU model whitelist. OptiScaler rewrites the GPU info returned by DXGI/Streamline within the game process, making it see your chosen native, 4090, or 5080M profile.

vs. Registry Rename

Registry renaming is a system-wide change. This tool by default only replaces the local proxy DLL in the game directory — it does not modify the system GPU registry.

Backup Strategy

Before installing, saves winmm.dll, OptiScaler.ini, the OptiScaler\ folder, and any old DLSSTweaks files, then writes a manifest.

Failure Handling

If the game fails to launch, restore the most recent backup first. If the option still doesn't appear, check the logs to confirm whether OptiScaler was loaded.

Core Project

GPU spoof core comes from OptiScaler. DLSSTweaks is only mentioned for legacy local file compatibility backup purposes — this tool does not use its DLSS scaling pipeline.
